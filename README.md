# TechHaven - Next-Gen Electronics

TechHaven is a modern, responsive e-commerce web application designed for browsing and purchasing consumer electronics.

## Features

- Dynamic product catalog
- Shopping cart functionality
- User authentication (Coming soon)
- Cloud database integration via Firebase

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **Styling:** Tailwind CSS
- **Backend/Database:** Firebase (Firestore & Auth)

## How to Run Locally

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
